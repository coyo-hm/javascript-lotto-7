# javascript-lotto-precourse

- [x] 로또 구입 금액 입력
  - [x] 구입 금액 단위는 1000원으로 나누어 떨어지지 않는 경우 예외 처리
  - [x] 숫자가 아니거나 음수일 경우 예외 처리
- [] 로또 1개 발행 함수: 중복되지 않는 6개의 숫자 뽑기 
- [] 발행 금액만큼 로또 발행
  - [] 발행 로또 출력(오름차순정렬)
- [] 당첨 번호 입력
  - [] 숫자 입력(1-45 사이의 숫자) 확인 => 예외 처리
  - [] 중복 번호 확인 => 예외 처리
  - [] 쉽표(,)로 구분 => 예외 처리
- [] 보너스 번호 입력
  - [] 숫자 입력(1-45 사이의 숫자) 확인 => 예외 처리
  - [] 중복 번호 확인 => 예외 처리
- [] 당첨 여부 확인
  - [] 당첨 내역 출력
  - [] 당첨 금액 계산
- [] 수익률 계산
